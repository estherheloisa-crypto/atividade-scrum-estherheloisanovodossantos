# Sistema de pedidos para um restaurante

Descrição:

Este projeto é um sistema simples para controlar pedidos de um restaurante. Nele, será possível criar pedidos, adicionar itens, ver os pedidos feitos e atualizar o status. O objetivo é ajudar na organização e no atendimento dos clientes, para mais rapidez e facilidade, tanto para cliente quanto para o restaurante.


Papéis do Scrum:

1. Product Owner: decide o que será feito no sistema.
2. Scrum Master: ajuda a organizar o trabalho e garantir que o Scrum seja seguido.
3. Time de Desenvolvimento: desenvolve o sistema.

Eventos do Scrum:

1. Sprint: período de tempo para fazer as tarefas.
2. Sprint Planning: planejamento do que será feito na sprint.
3. Daily Scrum: reunião rápida para ver como está o andamento.
4. Sprint Review: mostrar o que foi feito.
5. Sprint Retrospective: ver o que deu certo e o que pode melhorar.

Artefatos do Scrum:

1. Product Backlog: lista de tudo que o sistema precisa ter.
2. Sprint Backlog: tarefas escolhidas para a sprint.
3. Incremento: o que foi desenvolvido no final da sprint.

Product Backlog:

1. Criar pedido.
2. Adicionar itens ao pedido.
3. Listar pedidos.
4. Atualizar status do pedido.
5. Cancelar pedido.
6. Remover itens.
7. Calcular valor total.
8. Colocar nome do cliente.
9. Forma de pagamento.
10. Filtrar pedidos.

Prioridades:

1. Criar pedido.
2. Adicionar itens.
3. Listar pedidos.
4. Calcular valor total.
5. Atualizar status.
6. Cancelar pedido.
7. Remover itens.
8. Nome do cliente.
9. Forma de pagamento.
10. Filtrar pedidos.

Justificativa da organização da lista:
Escolhi essa ordem porque primeiro vem o mais importante, que é criar e ver os pedidos. Depois vêm funções para organizar melhor e melhorar o sistema.

Sprint Planning

Duração da Sprint: 2 semana

Itens escolhidos:

1. Criar pedido.
2. Adicionar itens.
3. Listar pedidos.


Sprint Backlog:

1. Fazer função de criar pedido.

Burndown:

O gráfico Burndown mostra quanto trabalho ainda falta fazer na sprint. Ele serve para acompanhar o progresso e ver se as tarefas vão terminar no tempo certo.
